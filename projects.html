Project: Color My Canvas

Course: CSP (Computer Science Principles)

Duration: Approximately 3 Weeks (Mid September 2018 - Early October 2018)

Objective: Using a template, create an app with your own design elements using MIT App Inventor.

Type of Project: Partner Project

For the MIT App Inventor Project, my partner and I chose to redesign this app called 'Colored Dots,' which only painted in dots and made us create and store different color variations, which was extremely difficult to use. We were provided with criteria to partner program an app with a detailed log as to how we evenly distributed our work. We took this project apart and rebuilt it so that it fit our necessary requirements. The newly created app now has a bar with prestored colors and a canvas that draws lines in any and all directions. The size of the paintbrush was made to be adjustable and we had two different eraser settings, one where it erased the entire canvas and another where where only a small section of the canvas where the finger is dragged would be erased. A separate button also provided instuctions to the user for an overall easy app experience. This project scored remarkably high because it followed the rubric. Throughout this project, I learned how to use advanced features on MIT App Inventor and how to troubleshoot in order to make one aspect of the project carry out various functions.
